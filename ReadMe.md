# 📝 To-Do List App

A simple and interactive **To-Do List Application** built with **HTML, CSS, and JavaScript**.  
This app allows users to add, mark as completed, and delete tasks. It also uses **Local Storage** to save tasks even after refreshing the page.

---

## 🚀 Features
- Add new tasks
- Mark tasks as completed ✅
- Delete tasks ❌
- Persistent data storage using **Local Storage**
- Responsive and clean UI
